System requirements:

  • Windows 2k, XP or newer.
  • Soundcard
  • Fairly fast computer for best framerates (my 1200mhz Athlon gets ~100 fps.

    • Keyboard commands:

  • Arrows - Rotate object
  • A, Z - Zoom in/out
  • V - Toggle Spheres/Dots
  • F - Show/hide FPS
  • X - Toggle auto rotation
  • D - Turn on Debug Mode
  • 1-4, 0 (when in debug mode) - Turn on specific debug mode.
  • ESC - Quit

    • Programming notes:

  • You should compile it for better frame rates.
  • There aren't very much comments, sorry!
